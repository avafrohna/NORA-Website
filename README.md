# NORA Website

This repository contains the early website for NORA.

The site is being built as a simple public-facing landing page to present:
- the problem NORA addresses
- the mission and value proposition
- team information
- credibility and validation
- contact details for pilot or collaboration interest

Current status: in development.

## Project structure

- `nora-site/` – website files
- `openspec/` – planning and specification documents

## Goals

The first version of the site is intended to be:
- clear
- credible
- lightweight
- easy to deploy publicly

## Deployment

This site is intended to be hosted online, not run only locally.

## Notes

NORA is currently in development, and the website content and structure may evolve as the project progresses.
